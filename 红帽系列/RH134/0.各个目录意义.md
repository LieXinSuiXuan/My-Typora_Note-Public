# 各个目录的意义

```powershell
/etc/group   查看组信息
/etc/passwed  储存用户的文件
/bin, 
/usr/bin,用户可执行命令文件
/usr/sbin, 管理员执行的命令
/usr/local/sbin  本地自定义软件安装目录

/media, /mnt 挂载点
/etc  配置目录
/tmp  临时文件
/boot  启动文件
/proc, /sys  内存状态
/var,/srv  网站数据
/lib,/usr/lib,/usr/local/lib  驱动程序
.bash_history 放置历史记录的文件
/etc/sysconfig/network-scripts   网络配置文件
/etc/fstab   可以设置磁盘自动挂载
```
